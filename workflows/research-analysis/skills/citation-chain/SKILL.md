# Step 3: The Citation Chain

From the loaded papers, identify the **3 concepts** that appear most frequently across different papers (mentioned by name, discussed, or developed).

## For Each Concept

Trace its intellectual evolution using **only information from these papers**:

### Concept Name: {name}

- **Origin:** Who first introduced or defined this concept (within these papers)? What was the original formulation?
- **Challenge:** Which papers challenged, criticized, or disputed it? How exactly?
- **Refinement:** Which papers modified, extended, or refined it? In what way?

### Current Status

Is this concept **established**, **contested**, or **still evolving** — based solely on these papers?

<RULES>
- Present each concept as a structured plan
- If a concept has no explicit criticism or refinement in these papers, state that directly — do not fabricate
- Use ONLY information from the loaded papers, no external knowledge
</RULES>

---

## Exit Criteria

- [ ] 3 most cross-referenced concepts identified
- [ ] Each concept has Origin, Challenge, and Refinement traced
- [ ] Current status assessed for each concept
- [ ] No information fabricated — gaps stated explicitly

---

## Anti-Rationalization Table

| Excuse | Reality |
|--------|---------|
| "All concepts are equally referenced" | Rank by frequency of appearance across papers. There is always a top 3. |
| "This concept has no challenges" | State that explicitly. Absence of criticism is itself a finding. |
| "I should add context from general knowledge" | No. Only the loaded papers. This traces evolution within this specific literature. |
